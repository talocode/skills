## Theme Application Rules

### Color Usage
- Primary: Main actions, links
- Secondary: Supporting elements
- Accent: Highlights, notifications
- Error: Errors, warnings
- Success: Confirmations, positive states

### Typography Rules
- Headings: Bold, clear hierarchy
- Body: Readable, 14px minimum
- Code: Monospace, clear contrast
- Labels: Uppercase, small, muted

### Spacing Rules
- Use consistent spacing scale
- Maintain visual rhythm
- Don't crowd elements
- Leave breathing room

## Theme Application Checklist

- [ ] Colors match product mood
- [ ] Typography is readable
- [ ] Spacing is consistent
- [ ] Interactions feel right
- [ ] Accessibility basics covered
- [ ] No visual clutter

## Notes

- Themes are guidelines, not strict rules
- Adapt to context
- Test on real devices
- Keep it simple
