# PolicyLane

**Agent policy gate** — allow/deny tool actions, match roles/tags, and redact secrets from payloads.

## Install

```bash
npm install -g @talocode/policylane
export TALOCODE_API_KEY=...
export TALOCODE_BASE_URL=https://api.talocode.site
```

## Routes

`/v1/policylane/*` on Talocode Cloud.
