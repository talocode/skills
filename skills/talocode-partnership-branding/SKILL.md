# talocode-partnership-branding

Create premium partnership / collaboration brand images in the Talocode style.

## Purpose

Generate sleek partnership announcement graphics for Talocode ecosystem brands and related projects.

This skill is for images like:

- ClipLoop × Talocode
- Codra × Talocode
- Tera × Talocode
- Tradia × Talocode
- WorkLane × Talocode
- Talocode Skills × Talocode
- Partner brand × Talocode
- Product integration announcement graphics
- Ecosystem relationship visuals
- launch/support/collaboration identity graphics

## Core visual style

The default visual language should match this approved direction:

- dark luxury tech aesthetic
- cinematic blurred black / charcoal / soft grey background
- subtle spotlight or fog-like light wash
- very clean composition with generous negative space
- centered horizontal alignment
- left brand icon/logo
- thin vertical divider in the middle
- right-side Talocode brand lockup
- crisp white Talocode wordmark
- high contrast
- modern premium open-source tooling vibe
- minimal, not noisy
- no mockup clutter
- no extra UI unless explicitly requested

## Standard composition

Preferred structure:

1. Left:
   partner/product logo or icon

2. Center:
   thin vertical divider line

3. Right:
   Talocode mark + wordmark

Default right-side brand treatment:
- Talocode angle-bracket mark: `<>`
- white Talocode wordmark
- geometric/futuristic clean type treatment

## Background rules

Use:
- blurred dark metallic / soft gradient background
- black to charcoal tones
- soft grey highlight areas
- subtle texture/grain allowed
- premium, smooth, elegant

Avoid:
- bright colorful backgrounds
- busy patterns
- cheap gradients
- random decorative elements
- glowing neon overload

## Logo handling rules

- Preserve the core identity of the partner logo/icon
- Keep layout clean and balanced
- Use vector-like clarity when possible
- Make logos visually harmonize without distorting identity
- If only an icon is available, use the icon cleanly
- If a wordmark is included, keep it crisp and minimal
- Do not invent false partnerships or affiliations

## Talocode-side rules

When Talocode appears on the right:
- use white symbol + white wordmark
- use the angle-bracket icon as the Talocode mark
- maintain premium spacing
- align logo and wordmark horizontally
- keep proportions balanced with the left-side logo

## Divider rules

- use a thin vertical divider line
- color: light grey / soft white
- keep it subtle
- enough spacing around divider for a premium feel

## Output intent

These graphics should feel like:
- a serious brand partnership
- a product integration reveal
- a polished announcement asset for X / social / launch posts
- a clean ecosystem relationship image

## Default aspect ratios

Use depending on context:

- 16:9 for wide promo image
- 1:1 for square social card
- 3:2 if a slightly cinematic card is preferred

Default:
- use 16:9 unless otherwise requested

## Prompt template

When generating, use this structure:

"Create a sleek premium partnership branding graphic in the Talocode style. Use a dark blurred luxury-tech background with soft grey lighting and lots of negative space. Place [Partner/Product Logo] on the left, a thin subtle vertical divider in the middle, and the Talocode brand mark and wordmark on the right in crisp white. Keep the composition minimal, elegant, modern, and high-contrast. Make it feel like a serious open-source tech partnership or ecosystem announcement. No extra clutter, no mockups, no random UI, no unnecessary text."
