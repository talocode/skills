# Templates

This directory contains markdown templates for Telegram posts.

## Available Templates

- `product-launch.md` — Product release announcement
- `changelog.md` — Version changelog
- `announcement.md` — General announcement

## Usage

Use the `telegram-ops template` command to generate posts from templates:

```bash
telegram-ops template --type product-launch --name "MyApp" --version "1.0.0"
```
