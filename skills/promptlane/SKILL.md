# PromptLane

**Prompt versioning and feedback patches** for copilots — capture thumbs-down and apply deterministic preference patches.

## Install

```bash
npm install -g @talocode/promptlane
export TALOCODE_API_KEY=...
export TALOCODE_BASE_URL=https://api.talocode.site
```

## Routes

`/v1/promptlane/*` on Talocode Cloud.
