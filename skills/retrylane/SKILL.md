# RetryLane

**Tool-call reliability for agents** — classify failures, plan retries with backoff/jitter, and trip circuit breakers.

## Install

```bash
npm install -g @talocode/retrylane
export TALOCODE_API_KEY=...
export TALOCODE_BASE_URL=https://api.talocode.site
```

## Routes

`/v1/retrylane/*` on Talocode Cloud.
