# talocode-remotion-demo (Legacy)

**This skill has been renamed to `talocode-video`.**

Use the new skill instead:

```bash
npx skills add talocode/skills/talocode-video
```

The new `talocode-video` skill covers:

- Remotion
- HyperFrames-style HTML video
- ffmpeg
- ClipLoop timeline engine
- Release demo videos
- Social videos
- Agent-authored video compositions

## Migration

If you have existing references to `talocode-remotion-demo`, update them to `talocode-video`.

The functionality is the same, but expanded to support more video generation approaches.

## Legacy Install

```bash
npx skills add talocode/skills/talocode-remotion-demo
```

This will install the compatibility wrapper that points to `talocode-video`.
