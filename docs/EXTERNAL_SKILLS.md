# External Skills Reference

This document references external skill libraries that Talocode agents can use.

## Taste Skill by Leonxlnx

**Repository**: https://github.com/Leonxlnx/taste-skill  
**Site**: https://tasteskill.dev  
**License**: MIT  
**Install**: `npx skills add Leonxlnx/taste-skill`

### What it is

Taste Skill is an external open-source skill library by Leonxlnx for improving AI-generated frontend design quality. It provides skills for:

- Frontend UI polish
- Brand kit generation
- Image-to-code conversion
- Redesign quality
- Layout, typography, spacing, motion

### When to use

Talocode agents can install and use Taste Skill when working on:

- Landing pages and redesigns
- Brand kits and visual identity
- Frontend polish and UI improvements
- Image-to-code conversions
- Anti-slop frontend generation

### Attribution

Taste Skill is not maintained by Talocode. Use the upstream repo for updates and attribution:

- GitHub: https://github.com/Leonxlnx/taste-skill
- Site: https://tasteskill.dev
- License: MIT (preserve attribution)

### Installation

```bash
npx skills add Leonxlnx/taste-skill
```

### Available Skills

- `taste-skill` - Core frontend taste skill
- `brandkit` - Brand kit generation
- `redesign-skill` - Redesign quality
- `image-to-code-skill` - Image to code conversion
- And more in the upstream repo

## Using External Skills

To use an external skill in Talocode projects:

1. Install the skill: `npx skills add <owner>/<repo>`
2. Reference it in your agent prompts
3. Follow the upstream license terms

## Notes

- Talocode does not own or maintain external skills
- Always check the upstream repo for updates
- Preserve attribution when using external skills
- Do not repackage external skills as Talocode-owned
